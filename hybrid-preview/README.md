# BoPeePs Hybrid Preview

This standalone preview reuses the repository's existing approved assets and leaves the current homepage unchanged.

Preview URL: `https://kurtuscobain.github.io/bopeeps-details-preview/hybrid-preview/`

- `index.html` contains the semantic homepage and quote form.
- `styles.css` provides the responsive red-and-black design, accessible focus states, and reduced-motion support.
- `script.js` powers service tabs, quote selections, SMS/email quote handoff, and mobile navigation.
